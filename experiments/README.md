# Experiments

This folder is for experiment scripts and notes that orchestrate reusable code from `src/libraries/` or `src/experiments/`.

Keep reusable implementation code in Git-tracked Python modules. Store generated games under `generated/games/`, counterexamples under `generated/counterexamples/`, and run outputs under `results/`.

SQLite stores metadata, commands, result summaries, paths, and git commit hashes. It does not store source-code blobs.
